# Blank

Minimal, Device agnostic & responsive barebone framework.

_In beta_

The main principles behind this are:

* Reusable
* Maintainable
* Predictable
* Scalable

### Browser support

* IE 8+
* Firefox
* Safari 6+
* Chrome
* Opera

### Build on top of these methodologies
* [OOCSS](https://github.com/stubbornella/oocss)
* [BEM](http://bem.info/method/)
