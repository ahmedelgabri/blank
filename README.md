# Blank CSS

This is a WIP mobile-first, responsive CSS framework to use as a starting point
in my projects.

_In beta_

The main principles behind this are:

* Reusable
* Maintainable
* Predictable
* Scalable

* [OOCSS](https://github.com/stubbornella/oocss)
* [SMACSS](http://smacss.com/)
* [BEM](http://bem.info/method/)
